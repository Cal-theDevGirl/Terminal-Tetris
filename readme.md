# Terminal Tetris

**Controls**
- Move     - ← ↓ →
- Drop     - Space
- Rotate   - ↑
- Pause    - p
- Quit     - q
- New Game - Enter

#### Play on mac

build it `PyInstaller tetris.py --onefile` and add *path* to the .bash_profile

`export PATH="/Users/your_name/tetris-terminal:$PATH"`

type in terminal and play :) `>tetris`
